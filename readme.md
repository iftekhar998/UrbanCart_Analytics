## рџ“Њ Project Overview

**UrbanCart** is a growing online retail shop operating across multiple cities in Bangladesh. The management team wanted to become more data-driven to increase revenue, improve product recommendations, understand customer behavior, and optimize inventory and payment operations.

As the Data Analyst on this project, I analyzed UrbanCart's transactional data using **PostgreSQL** and delivered actionable insights to support business strategy, marketing decisions, and product placement.

## рџљЂ How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/iftekhar998/UrbanCart_Analytics.git
   cd UrbanCart_Analytics
   ```

2. **Set up the database**  
   Create a new PostgreSQL database and run the schema file:
   ```sql
   CREATE DATABASE urbancart;
   \c urbancart
   \i schema/create_tables.sql
   ```

3. **Load the data**  
   Import each CSV file into the corresponding table using `COPY` or pgAdmin's import tool.

4. **Run the queries**  
   Open any `.sql` file from the `queries/` folder and execute it against the `urbancart` database.
